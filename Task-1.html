Simple Webpage Using HTML , CSS , JavaScript

HTML (HyperText Markup Language)

HTML is the structure of a webpage.It defines what content appears, like headings, paragraphs, images, buttons, etc.

CSS (Cascading Style Sheets)

CSS is the design of a webpage.It controls how the content looks, like colors, fonts, layout, size, and spacing.

JavaScript (JS)

JavaScript is the behavior of a webpage.It makes the page interactive, like responding to button clicks, showing alerts, animations, etc.
